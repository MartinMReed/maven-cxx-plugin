SETUP FOR ECLIPSE STANDARD
* Download the ActionScript Dev Tools (ASDT):
** Select the stable release (rather than a beta)
** http://aseclipseplugin.sourceforge.net/updates
* Adobe AIR SDK: http://www.adobe.com/products/air/sdk
** /tools/adobe/adobe-air-sdk-2.5
* Tablet SDK: http://us.blackberry.com/developers/tablet/devresources.jsp
** select /tools/rim as the install location (will install as /tools/rim/blackberry-tablet-sdk-0.9.1)
** select /tools/adobe/adobe-air-sdk for the AIR SDK location
* VMWare Fusion: http://downloads.vmware.com/d/info/desktop_downloads/vmware_fusion_for_the_mac/3_0
* Tablet Simulator: http://us.blackberry.com/developers/tablet/devresources.jsp

SETUP FOR ECLIPSE + FLASH BUILDER
* Eclipse 3.5: http://www.eclipse.org/downloads/packages/eclipse-ide-java-developers/galileosr2
** /tools/eclipse/eclipse-3.5-carbon
* Adobe AIR SDK: http://www.adobe.com/products/air/sdk
** /tools/adobe/adobe-air-sdk-2.5
* Eclipse 3.5 Flash Builder Plugin: https://www.adobe.com/cfusion/tdrc/index.cfm?product=flash_builder
** /tools/adobe/adobe-flash-builder-plugin
* Tablet SDK: http://us.blackberry.com/developers/tablet/devresources.jsp
** select /tools/adobe/adobe-flash-builder-plugin for the Flash Builder location
** select /tools/eclipse/eclipse-3.5-carbon for the Eclipse location
** select /tools/adobe/adobe-air-sdk for the AIR SDK location
* VMWare Fusion: http://downloads.vmware.com/d/info/desktop_downloads/vmware_fusion_for_the_mac/3_0
* Tablet Simulator: http://us.blackberry.com/developers/tablet/devresources.jsp
